Postman 小技巧

在拿到 token 後，跟 api 發出一個請求都要夾帶 token，這裡可以手動貼上，也可以在 test 的部份貼上一段程式碼，將 token 存成變數，之後在 header 裡 Authorization 直接對應 Bearer {{AUTH_TOKEN}} ，Postman 就會帶入剛剛拿到的 token 了。

坑點

在測試 cors 情境的時候，遇到 server 回傳的 response header 一直沒有夾帶　Access-Control-Allow 系列的參數，以為是設定有錯，追了原始碼才確認不是 PHP 段的問題。最終發現，問題出在 server 安裝的 pagespeed module，所以在/etc/apache2/sites-available/000-default.conf和/etc/apache2/sites-available/000-default-le-ssl.conf 將 api server 的 pagespeed 關閉，cors 就能正常使用了。這個 issue 可以參考
Missing CORS headers in response from PHP/Apache2 。
